Glossary

General Engineering
Robust Design
A design that provides the intended functions, in stated conditions, for the specified time, in the presence of noise.
Functional Requirement
Functionality a product must deliver to satisfy customer needs.
Verification Test
Testing done to ensure a product meets customer's requirements.
Development Test
Testing done to explore the design space in order to understand limits and relationships.
Critical Parameter Management
A process for ensuring parameters that are critical to product performane are clearly defined and accounted for
NUD
Requirements that are " NEW, UNIQUE, AND DIFFICULT" and therefore warrent additional time and attention to detail to prevent quality escapes.
DFMEA
Design Failure Modes and Effects Analysis - a process for identification of potential failure modes and methods for mitigation.
Statistics Terms
Normality
The property that determines how well a set of data fits a normal distribution. Knowing this aids in selecting the best statistical tools for further analysis.
P value
The probability that a null hypothesis has been satisfied and therefore cannot be rejected.
Model
A fabrication intended to predict output parameters based on input parameters. Used regularly in development as an analytical tool that can be used in lieu of physical testing.
Mean
The statistical property that describes the central tendancy of a data set (assumed to be normally distributed). Usually thought of as the " avarage".
Median
The statistical property that describes the central tendancy of a data set (assumed not to be normally distributed).
Standard deviation
The statistical property that describes the spread of a data set (assumed to be normally distributed). The simbol σ is used to designate standard deviation. Calculated as the square root of the variance. One nice property of σ is that the unit of measure for σ is the same as that of the data in the set and is therefor more intuitive.
Variance
The average of the squared difference from the mean of a data set. One nice property of variance is that they can be summed across data sets to determine vafriance for the sum of the sets.
Reliability Terms
Stress/Strength relationship
A property that can be used to evaluate reliability based on product strengths (generally measured) against stresses applied in teh field (generally assumed).
Failure Mode
A way that a product or part can fail in the field. Quality escapes can be reduced by early identification and mitigation of failure modes.
Failure Site
The spicific design feature whare a known failure mode occurs in a product or part. These should be clearly identified and maintained in DFMEAs.
